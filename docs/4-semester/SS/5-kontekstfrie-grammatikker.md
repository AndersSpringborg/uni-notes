 ## Definition

En kontekstfri grammatik er en 4-tupel
$$
(V,\Sigma,)
$$

| Symbol   | Betydning                    |
| -------- | ---------------------------- |
| V        | Endelig mængde af variable   |
| $\Sigma$ | Endelig mængde af terminaler |
| R        | Endelig mængde af regler     |
| S        | Startvariabel $S\in V$       |
